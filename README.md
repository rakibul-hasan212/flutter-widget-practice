# 📱 Flutter Widget Practice Projects

This repository contains a collection of small Flutter projects built to practice and master important Flutter widgets.

## 🎯 Goal

The main goal of this repository is to:

* Learn Flutter UI development
* Understand how widgets work in real-world scenarios
* Build strong fundamentals using hands-on practice

* Icons
* GridView
* Drawer
* SnackBar
* TextField & Form
* Buttons
* AppBar
* Container & Styling

## 🧠 What I Learned

* Layout system (Row, Column, Expanded)
* State management basics (setState)
* Form validation
* UI structuring and reusable components
